# ChatRoom-using-node.js
simple chat room handled users and message in back and front-end with socket.io as websocket in node.js
## Dependencies
```
    express   : 4.17.1
    moment    : 2.29.1
    socket.io : 4.1.3
Dev-Dependencies : 
    nodemon   : 2.0.12
```
## To run the app
```
npm install
nodemon server 
```
Url : http://localhost:3000
