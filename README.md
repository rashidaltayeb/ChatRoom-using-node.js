# ChatRoom-using-node.js
simple chat room handled users and message in back and front-end with socket.io as websocket in node.js
## package.json
```
{
  "name": "chat-room",
  "version": "1.0.0",
  "description": "",
  "main": "index.js",
  "scripts": {
    "start": "node server",
    "dev":"nodemon server"
  },
  "author": "",
  "license": "ISC",
  "dependencies": {
    "express": "^4.17.1",
    "moment": "^2.29.1",
    "socket.io": "^4.1.3"
  },
  "devDependencies": {
    "nodemon": "^2.0.12"
  }
}

```
## To run the app
```
npm install
nodemon server 
```
Url : http://localhost:3000
